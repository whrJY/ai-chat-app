# AI Chat App

一个使用 React + TypeScript + Tailwind CSS 构建的现代化 AI 聊天应用。

## 功能特性

- 🎨 现代化 UI 设计
- 🌙 主题切换（亮色/暗色/系统）
- 💬 实时聊天体验
- ⚡ TypeScript 类型安全
- 📱 响应式设计
- 🔄 错误处理和重试机制

## 快速开始

```bash
# 克隆项目
git clone https://github.com/whrJY/ai-chat-app.git
cd ai-chat-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React
